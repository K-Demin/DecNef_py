Summary

This sample project demonstrates how to use the BIOPAC Hardware API in C# to create a simple biofeedback application.

Figure 1: Make the red circle smaller than the green circle by relaxing.

Introduction

Biofeedback is a method of training subjects to gain some control over physiological response. This usually involves monitoring the subject's physiological signal and presenting some visual, auditory, or physical stimulus. The stimulus reacts according to the subject's physiological response.

Galvanic Skin Response (Electrodermal Activity)

One of the most common methods of training a person to relax is to monitor eccrine activity. Electrodermal Activity is the measure of skin's conductiveness. As a person's stress level increases, the body begins to perspire. Increased eccrine activity (perspiration) increases the skin's conductiveness.

Design

This basic Biofeedback system is designed to train a subject to relax. The subject is connected to the MP device via the TSD203 and EDA100C. Amplifier settings can be found in the documentation that shipped with your MP System. The subject is asked to keep the red circle inside the green circle. The green circle represents the desired relaxation state and the red circle represents the subject's stress level.

Implementation

As soon as the application is launched, it calls on initAcquistionUnit(). Using the Hardware API, this function configures the MP device to acquire on channel 1 at 500 Hz. It also starts the acquisition.

The form contains a timer called updateTimer that has a tick interval of 500 msec. The tick event is handled by the UpdateScreen() function which redraws the circles. The red circle's diameter is determined by the function called GetCircleDiameter().

GetCircleDiameter()

The GetCircleDiameter() calls the Hardware API function getMostRecentSample(). Since EDA does not drastically change within a short period of time, polling the MP device for the most recent sample will suffice. Before GetCircleDiameter() returns, the EDA voltage level is scaled; zero volts maps to 50 pixels and 10 volts maps to 600 pixels.

Notes

  • Calls to the API method connectMPDev() need to be modified to reflect your MP device configuration
  • MPDevImports.cs is the C# wrapper for the Hardware API's methods
